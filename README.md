#  Smart Budget Utilization Monitoring System

##  Overview

A Python-based application that analyzes budget vs expenditure, predicts future utilization, and suggests fund reallocation using machine learning and a Streamlit dashboard.

##  Features

* Budget vs expenditure analysis
* Under-utilization detection
* Utilization prediction (Linear Regression)
* Fund reallocation suggestions
* Interactive Streamlit dashboard

##  Run the Project

```bash
pip install -r requirements.txt
streamlit run dashboard/app.py
```

##  Output

Predictions are saved to:

```
data/predictions.csv
```

##  Tech Stack

Python, Pandas, Scikit-learn, Streamlit
